📐 Formelsammlung ZP10

Formelsammlung Mathematik – Zentrale Prüfungen 10
Anforderungsniveau MSA (NRW)

Ebene Figuren
Quadrat A = a · a = a²
u = 4 · a
Rechteck A = a · b
u = 2·a + 2·b
Dreieck A = (g · h) / 2
u = a + b + c
Parallelogramm A = g · h
u = 2·a + 2·b
Trapez A = (a + c) / 2 · h
u = a + b + c + d
Kreis d = 2·r
A = π · r²
u = 2 · π · r = π · d
Geometrische Körper
Würfel V = a³
O = 6 · a²
Quader V = a · b · c
O = 2·(a·b + b·c + c·a)
Prisma V = G · hk
M = u · hk
O = 2·G + M
Zylinder G = π · r²
V = G · hk = π · r² · hk
M = u · hk = 2·π·r · hk
O = 2·G + M
Pyramide V = ⅓ · G · hk
Kegel V = ⅓ · π · r² · hk
M = π · r · s s = Seitenlänge (Mantellinie)
Kugel V = 4/3 · π · r³
O = 4 · π · r²
Satz des Pythagoras
Satz des Pythagoras a² + b² = c² c = Hypotenuse (längste Seite, gegenüber dem rechten Winkel)
Kathetensatz a² = c · p
b² = c · q p, q = Hypotenusenabschnitte
Höhensatz h² = p · q
Trigonometrie
Im rechtwinkligen Dreieck: sin(α) = Gegenkathete / Hypotenuse
cos(α) = Ankathete / Hypotenuse
tan(α) = Gegenkathete / Ankathete
Lineare Funktionen
Allgemeine Form y = m·x + b m = Steigung, b = y-Achsenabschnitt
Steigung berechnen m = (y₂ - y₁) / (x₂ - x₁) Steigungsdreieck: m = Δy / Δx
Nullstelle x₀ = -b / m
Quadratische Funktionen
Normalform f(x) = ax² + bx + c
Scheitelpunktform f(x) = a·(x - d)² + e Scheitelpunkt S(d | e)
p-q-Formel (Nullstellen) x₁,₂ = -(p/2) ± √((p/2)² - q) für x² + px + q = 0
Scheitelpunkt aus Normalform xS = -b / (2a)
yS = f(xS)
Potenzen und Wurzeln
am · an = am+n
am / an = am-n
(am)n = am·n
a⁰ = 1
a-n = 1 / an
Binomische Formeln (a + b)² = a² + 2ab + b²
(a - b)² = a² - 2ab + b²
(a + b)(a - b) = a² - b²
Wachstum und Zinsrechnung
Zinseszins K(n) = K₀ · (1 + p/100)n K₀ = Anfangskapital, p = Zinssatz, n = Jahre
Exponentielles Wachstum f(t) = a · bt a = Anfangswert, b = Wachstumsfaktor (b > 1: Wachstum, b < 1: Zerfall)
Lineares Wachstum f(t) = a + m · t a = Anfangswert, m = konstante Änderungsrate
Stochastik
Laplace-Wahrscheinlichkeit P(E) = Anzahl günstiger Ergebnisse / Anzahl möglicher Ergebnisse
Gegenereignis P(Ā) = 1 - P(A)
Pfadregeln (Baumdiagramm) Pfadmultiplikation: P(A∩B) = P(A) · P(B)
Pfadaddition: P(A∪B) = P(Pfad₁) + P(Pfad₂) + ...
Erwartungswert E(X) = x₁·P(x₁) + x₂·P(x₂) + ... + xn·P(xn)
Relative Häufigkeit h(E) = H(E) / n H(E) = absolute Häufigkeit, n = Gesamtanzahl
Prozentrechnung
Prozentwert: W = G · p/100
Grundwert: G = W · 100/p
Prozentsatz: p = W/G · 100
