# ⚡ Schnelleinstieg - ZP10 mit Azure (5 Minuten)

**Für Lehrer, die sofort starten möchten.**

---

## Schritt 1: Login-Seite öffnen (30 Sekunden)

Öffnen Sie im Browser:
```
https://yourschool.de/zp10/azure/zp10-login-azure.html
```

(Ersetzen Sie `yourschool.de` mit Ihrer Domain)

---

## Schritt 2: Azure konfigurieren (2 Minuten)

Falls Sie Azure noch nicht eingerichtet haben:

1. **Klicken Sie:** "⚙️ Azure konfigurieren" (unten auf der Seite)
2. **Geben Sie 3 Werte ein:**

   ```
   Client ID:     [Von Ihrem IT-Administrator]
   Tenant ID:     [Von Ihrem IT-Administrator]
   API Base URL:  https://api.schulname.de (oder ähnlich)
   ```

3. **Klicken Sie:** "✓ Speichern"

**Wo bekomme ich diese Werte?**

Fragen Sie Ihren IT-Administrator oder schauen Sie in die vollständige Anleitung:
→ Datei `SETUP-AZURE.md` Schritt 5

---

## Schritt 3: Anmelden (1 Minute)

1. **Klicken Sie:** "Mit Schulkonto anmelden"
2. **Melden Sie sich an** mit Ihrem Schulkonto
3. **Bestätigen Sie** Berechtigungen (falls gefragt)

Sie sehen jetzt:
- ✅ Ihren Namen
- ✅ Ihre E-Mail
- ✅ Ihre Rolle (Lehrer/Schüler)

---

## Schritt 4: Dashboard öffnen (1 Minute)

Nach dem Login:

- **Lehrkräfte:** Klicken Sie "Zum Lehrer-Dashboard"
- **Schüler:** Klicken Sie "Zum Schüler-Hub"

Sie sind fertig! 🎉

---

## Was passiert jetzt?

### Als Lehrkraft:

Im Lehrer-Dashboard sehen Sie:
- 📊 **Übersicht**: Klassen, Schüler, Tests
- 👥 **Klassen**: Liste Ihrer Schulklassen
- 🧑 **Schüler**: Einzelne Schüler und ihre Ergebnisse
- 🔥 **Heatmap**: Grafik wer welches Modul wie gut beherrscht
- ⚠️ **Fehlvorstellungen**: Häufige Fehler in der Klasse
- 🎯 **Kompetenzen**: Welche Fähigkeiten sind schwach
- 📝 **Notenprognose**: Geschätzte Schulnoten nach NRW-Vorgaben
- 💡 **Förderempfehlungen**: Automatische Tipps pro Schüler
- 📥 **Export**: Daten als CSV/PDF speichern

### Als Schüler:

Im Schüler-Hub sehen Sie:
- 📚 Alle 13 Mathe-Module
- ✏️ Test starten
- 📊 Eigene Ergebnisse
- 🏆 Fortschritt

---

## Keine Azure? Offline-Modus!

Falls Sie Azure nicht haben/wollen:

1. **Klicken Sie:** "Offline weiterarbeiten"
2. **Arbeiten Sie offline** - alles wird lokal gespeichert
3. **Später online?** → Automatische Synchronisierung

**Offline funktioniert immer!** ✅

---

## Häufige Probleme

### "Anmeldefehler"
→ Überprüfen Sie, ob Sie das richtige Schulkonto nutzen

### "Client ID ist falsch"
→ Fragen Sie Ihren IT-Administrator

### "Daten werden nicht gespeichert"
→ Überprüfen Sie Internetverbindung

### "Ich sehe das Dashboard nicht"
→ Aktualisieren Sie die Seite (F5)

---

## Nächste Schritte

1. **Schüler einladen**
   - Link teilen: `zp10-hub.html`
   - Oder QR-Code ausdrucken

2. **Klassen einrichten** (im Dashboard)
   - Klassen hinzufügen
   - Schüler zuordnen

3. **Module starten lassen**
   - Schüler machen Tests
   - Sie sehen Ergebnisse in Echtzeit

4. **Regelmäßig checken**
   - Dashboard öffnen
   - Schülerfortschritt anschauen
   - Förderempfehlungen umsetzen
