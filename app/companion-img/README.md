# Companion-Bilder

Lege hier die generierten Bilder ab. Das System erkennt sie automatisch anhand des Dateinamens.

## Dateinamens-Konvention

```
{typ}-{stufe}.png
```

### Ritter (knight)
- `knight-1.png` — Lehrling
- `knight-2.png` — Geselle
- `knight-3.png` — Adept
- `knight-4.png` — Veteran
- `knight-5.png` — Champion
- `knight-6.png` — Meister

### Tiere
- `fox-1.png` bis `fox-6.png` — Fuchs
- `owl-1.png` bis `owl-6.png` — Eule
- `dragon-1.png` bis `dragon-6.png` — Drache
- `dolphin-1.png` bis `dolphin-6.png` — Delfin
- `wolf-1.png` bis `wolf-6.png` — Wolf
- `phoenix-1.png` bis `phoenix-6.png` — Phoenix

## Technische Anforderungen
- **Format:** PNG mit transparentem Hintergrund
- **Groesse:** ca. 400x500px (Hochformat, 4:5)
- **Charakter:** Mittig, leicht nach links gedreht (3/4-Ansicht)
- **Stil:** Cute RPG / Cartoon, einheitlich ueber alle Bilder

## Gesamt: 42 Bilder
- 6 Ritter-Stufen
- 6 Tiere x 6 Stufen = 36 Tier-Bilder

Wenn ein Bild fehlt, wird automatisch das SVG-Fallback angezeigt.
Du kannst also schrittweise Bilder hinzufuegen.
