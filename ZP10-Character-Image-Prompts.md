# ZP10 Character Image Prompts
## Companion Creatures for the ZP10 Math Diagnostic Tool

> **Target**: DALL-E 3 and Midjourney v6
> **Format**: Transparent PNG, ~400x500px, portrait orientation
> **Audience**: German 10th graders (15-16 years old)
> **Style**: Modern mobile game mascot, clean vector-like lines, cute RPG companion aesthetic. Not anime/chibi, not childish. Think Duolingo meets RPG companion.

---

## Filename Convention

```
{path}-{animal}-{stage}-{mood}.png
```

Examples:
- `knight-stage1-normal.png`
- `fox-stage3-sleepy.png`
- `owl-stage6-normal.png`

---

## Style Anchor (use at the start of every prompt)

**DALL-E prefix:**
> Digital illustration of a cute fantasy companion character, modern mobile game mascot style, clean vector-like lines, vibrant colors, transparent background, centered composition, 3/4 view facing slightly left, portrait orientation 4:5 aspect ratio.

**Midjourney prefix:**
> Cute fantasy companion character, modern mobile game mascot style, clean vector illustration, vibrant colors, isolated on pure white background, centered, 3/4 view facing slightly left, portrait format

**Midjourney suffix (always append):**
> --ar 4:5 --style raw --no background, shadow, floor, ground

# GENERATION TIPS

1. **Consistency**: Generate all stages of one character in a single session if possible. For DALL-E, use the same seed if the API supports it. For Midjourney, use `--seed [number]` with the same seed across stages.

2. **Transparent PNG**: DALL-E 3 natively supports transparent backgrounds. For Midjourney, the white background will need to be removed in post-processing (use remove.bg or Photoshop).

3. **Post-processing checklist**:
   - Remove any background artifacts
   - Ensure character is centered in a 400x500 canvas
   - Save as PNG-24 with transparency
   - Verify character reads well at 200px width (widget size)
   - Create a 2x version (800x1000) for high-DPI displays

4. **Batch workflow suggestion**:
   - Start with Fox (most recognizable, good baseline for style)
   - Then Owl and Dragon (most visually distinct)
   - Then Knight (most complex with armor details)
   - Then Wolf, Dolphin, Phoenix
   - Generate all "normal" moods first, then "sleepy" variants

5. **Midjourney specific**: After generating on white, run through background removal. Consider `--chaos 0` for more predictable results across a series.

6. **Color accuracy**: The hex colors are guide colors. The actual rendering will vary but should stay in the correct color family. If needed, color-correct in post.
